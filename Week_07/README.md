13课时作业 100万数据插入
1.单条数据循环插入
平均一分钟插入1000条
1000000条数据理论需要1000分钟=16h(和服务器性能也有一定的关系)

2.通过jdbc批量多事务提交（参见如下类：MysqlLearn）
如果1000000条数据，一次批量插入，sql太长，sql语法错误；
分多次事务，比如2次，5次，10次；通过实验，2次事务处理，每次批量处理500000条数据
，最快使用14s.

3.后续实验，如何引入线程池执行，效率是否可能更快。
